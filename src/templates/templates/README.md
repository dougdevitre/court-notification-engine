# Notification Templates

This directory contains pre-built notification templates for common court events.

## Planned Templates

| Template ID | Description | Status |
|-------------|-------------|--------|
| `hearing-reminder-7day` | 7-day hearing reminder | Planned |
| `hearing-reminder-3day` | 3-day hearing reminder | Planned |
| `hearing-reminder-1day` | 1-day hearing reminder | Planned |
| `hearing-reminder-today` | Same-day hearing reminder | Planned |
| `deadline-approaching` | Filing deadline approaching | Planned |
| `filing-received` | Court accepted your filing | Planned |
| `case-update` | General case status update | Planned |
| `payment-due` | Court fee or fine due | Planned |
| `missed-hearing` | Recovery — missed hearing | Planned |
| `missed-deadline` | Recovery — missed deadline | Planned |

## Writing Guidelines

- Write at a 6th-grade reading level
- No legal jargon — use plain language
- Include action steps ("What to do next")
- Always include the date, time, and location
- Keep SMS versions under 160 characters
- Include contact information for help
